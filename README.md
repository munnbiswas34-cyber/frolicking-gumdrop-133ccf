# INAYA ENTERPRISES — Currency Exchange Website

A professional currency exchange landing page for **INAYA ENTERPRISES**, specializing in USD, BDT (Bangladeshi Taka), and INR (Indian Rupee) transactions.

## Features

- **Live Exchange Rates Table** — Displays buy/sell rates for all six currency pairs (USD↔BDT, USD↔INR, INR↔BDT)
- **Interactive Currency Converter** — Real-time calculation using displayed rates with swap functionality
- **Services Showcase** — Cash exchange, remittance, wire transfer, business FX, mobile banking, consultation
- **About Section** — Company history, credentials, and key stats
- **Client Testimonials** — Social proof from real customer personas
- **Contact Form** — Service inquiry form with success state
- **Responsive Design** — Mobile-first layout with hamburger menu

## Tech Stack

- **Framework:** TanStack Start (React + Vite)
- **Styling:** Inline styles + Tailwind CSS v4
- **Icons:** Font Awesome 6 (CDN)
- **Fonts:** Inter (Google Fonts)
- **Hosting:** Netlify

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

To use Netlify features locally (functions, edge, etc.):

```bash
netlify dev --port 8889
```
